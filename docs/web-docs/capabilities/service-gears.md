Service gears (`gears/`) are business gears shipped as working examples you can run, read, and reuse.

| Gear | What it does | Status |
| --- | --- | --- |
| **File Parser** | Parse DOCX/PPTX/PDF/Markdown/HTML/text; extract text & metadata | ✓ |
| **Credentials Store** | Plugin-based secret storage with zeroize/redaction and tenant scoping | ✓ |
| **Mini Chat** | Minimal chat engine (conversations + messages) for examples/testing | ✓ |
| **Simple User Settings** | Per-user/tenant configuration with schema validation | ✓ |

A larger catalog of GenAI and platform service gears (Chat Engine, LLM Gateway, Models/Prompts/Agents registries, Serverless, Events Broker, Durable Objects, Jobs Manager, and more) is **designed but not yet implemented** — see [Status and roadmap](../status-and-roadmap/).
